You are running with the OrbCode plugin by MatterAI. All inference calls are routed through the MatterAI API for optimization.

Available OrbCode commands:
- `/orb-login` — Authenticate with MatterAI
- `/orb-logout` — Clear stored credentials
- `/orb-status` — Check authentication and proxy status
- `/orb-update` — Update the OrbCode plugin
- `/orb-setup` — Initial setup (proxy + environment configuration)
